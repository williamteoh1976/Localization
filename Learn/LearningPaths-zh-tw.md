## [SQL Server 升級](https://docs.microsoft.com/zh-tw/learn/paths/sql-server-2017-upgrades)
#### 學習途徑 - 3 小時 18 分鐘 - 5 單元

Modules:
- 升級 SQL Server 簡介

- 使用 SQL Server 查詢調整小幫手
